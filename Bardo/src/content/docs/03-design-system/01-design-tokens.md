# Design Tokens

Definição dos tokens (cores, espaçamentos, tipografia) e guideline para componentes reutilizáveis.

- Tokens: cores, espaçamentos, sombras
- Tipografia: `Space Grotesk` como fonte principal
- Componentes: Card, Button, Inputs
